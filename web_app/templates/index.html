Quick Start

Run GMC code faster and more intuitively through a brand-new dynamic UI. Generate interactive 3D correlation surfaces with one click to explore the internal consistency of IQA models globally.

Guidelines

Introduction

GMC provides a more informative and reliable evaluation paradigm for IQA models. This web console allows you to pass CLI arguments via a form, automatically execute the backend Python script, and view the analytical results in the 3D Viewer without refreshing the page.

Operation Steps

  1. Select Control Panel from the left navigation bar.
  2. Fill in the CSV data file path, e.g., ./data/example.csv.
  3. Set the subjective score (MOS Column) and prediction score (Prediction Columns) according to your data.
  4. (Optional) Adjust parameters such as the number of sampling points (Samples) and evaluation method (Method).
  5. Click "Run Computation", and enter the 3D Viewer to preview the rendered surface after execution is complete.
